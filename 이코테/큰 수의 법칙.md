# 📖 큰 수의 법칙(p.92)

## 📌 문제 정의
큰 수의 법칙은 다양한 수로 이루어진 배열이 있을 때 주어진 수들을 m번 더하여 가장 큰 수를 만드는 법칙이다.
단, 배열의 특정한 인텍스(번호)에 해당하는 수가 연속해서 k번을 초과하여 더해질 수 없는 것이 이 법칙의 특징이다.

예를 들어, 순서대로 2, 4, 5, 4, 6으로 이루어진 배열이 있을 때 m이 8이고, k가 3이라고 가정한다면,
특정 인덱스의 수가 연속해서 세 번까지만 더해질 수 있어 6 + 6 + 6 + 5 + 6 + 6 + 6 + 5인 46이 된다.
단, 서로 다른 인덱스에 해당하는 수가 같은 경우에도 서로 다른 것으로 간주한다.
예를 들어, 순서대로 3, 4, 3, 4, 3으로 이루어진 배열이 있을 때 m이 7이고, k가 2라고 가정한다면, 2번째 원소에 해당하는 '4'와
4번째 원소에 해당하는 4르ㅜㄹ 번갈아 두 번씩 더하는 것이 가능하다.
따라서, 4 + 4 + 4 + 4 + 4 + 4 + 4인 28이 도출된다.

</br>

### 입력 조건
- 첫째 줄에 N(2 <= N <= 1,000), M(1 <= M <= 10,000), K(1 <= K <= 10,000)의 자연수가 주어지며, 각 자연수는 공백으로 구분한다.
- 둘째 줄에 N개의 자연수가 주어진다. 각 자연수는 공백으로 구분한다. 단, 각각의 자연수는 1 이상 10,000 이하의 수로 주어진다.
  입력으로 주어지는 K는 항상 M보다 작거나 같다.

### 출력 조건
- 첫째 줄에 큰 수의 법칙에 따라 더해진 답을 출력한다.

</br>

## 📌 나의 풀이

~~~python
n, m, k = map(int, input().split())
arr = list(map(int, input().split()))
arr.sort()

first = arr[len(arr)-1]   #가장 큰수
second = arr[len(arr)-2]  # 두번째 큰수

group = [first] * k + [second]  #즉 그룹의 길이는 k+1과 같음

answer = group * (m // len(group)) + group[: m % len(group)]  # (m // len(group)) 묶음 갯수
print(sum(answer))
~~~

### 💡 문제 분석

🔹 m 횟수만큼 큰 수를 더하기만 하면 된다. 단, 특정 요소를 k번까지만 연속가능하다.
    - 즉, 두번째 큰 수를 중간에 더해주면서(번갈아 가면서) 사용해야 조건 만족 가능
    
~~~
예) arr = [1,2,3]  m = 5, k = 4 이라면 3 + 3 + 3 + 3 + 3 이 아니라 3 + 3 + 3 + 3 + 2가 맞음 (k번까지 연속가능)
~~~

🔹 배열 내에 값이 같더라도 인덱스 번호가 다르면 다른 요소로 취급
