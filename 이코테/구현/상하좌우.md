# 📖 이코테 구현 예제(4-1)
## 📌 문제 정의
여행가 A는 N x N 크기의 정사각형 공간 위에 서 있다. 이 공간은 1 x 1 크기의 정사각형으로 나누어져 있다.</br>
가장 왼쪽 위 좌표는 (1,1)이며, 가장 오른쪽 아래 좌표는(N x N)에 해당한다.</br>
여행가 A는 상, 하,좌, 우 방향으로 이동할 수 있으며, 시작 좌표는 항상 (1,1)이다.</br>
우리 앞에는 여행가 A가 이동할 계획이 적힌 계획서가 놓여 있다.</br>

계획서에는 하나의 줄에 띄어쓰기를 기준으로 하여 L, R, U, D 중 하나의 문자가 반복적으로 적혀 있다. 각 문자의 의미는 다음과 같다.

- L : 왼쪽으로 한 칸 이동
- R : 오른쪽으로 한 칸 이동
- U : 위로 한 칸 이동
- D : 아래로 한 칸 이동

이때 여행가 A가 N x N 크기의 정사각형 공간을 벗어나는 움직임은 무시된다.</br>
예를 들어(1,1)의 위치에서 L 혹은 U를 만나면 무시된다.</br>

### 입력
- 첫째 줄에 공간의 크기를 나타내는 N이 주어진다 (1 <= N <= 100)
- 둘째 줄에 여행가 A가 이동할 계획서 내용이 주어진다. (1 <= 이동 횟수 <= 100)

### 출력
- 첫째 줄에 여행가 A가 최ㅏ종적으로 도착할 지점의 좌표 (X,Y)를 공백을 기준으로 구분하여 출력한다.

### 테스트 예시
|입력|출력|
|---|---|
|5</br>R R R U D D|3 4|
</br>

## 📌 나의 풀이
~~~python
n = int(input())
move = list(map(str, input().split()))

dx = [0,0,-1,1] # LRUD
dy = [-1,1,0,0]
move_type = ['L', 'R', 'U', 'D']

x, y = 1, 1

for i in move:
    for j in range(len(move_type)):
        if i == move_type[j]: #예상 움직임 좌표 구하기
            nx = x + dx[j]
            ny = y + dy[j]

    # 지금 구한 좌표로 업데이트 할지 말지 결정하기
    if nx < 1 or ny < 1 or nx > n or ny > n:
        continue
    else:
        x, y = nx, ny #예상 움직임 좌표로 현재 좌표 업데이트!

print(x,y)
~~~

### 💡문제 분석
- 시뮬레이션 문제는 미리 dx, dy를 미리 셋팅
- 예상 움직임 좌표 구하기
- 범위가 벗어났을 경우에는 continue, 벗어나지 않았다면 예상 좌표로 **현재 좌표 업데이트!**
