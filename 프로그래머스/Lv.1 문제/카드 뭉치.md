# 📖 프로그래머스 Lv.1
## 📌 문제 정의
코니는 영어 단어가 적힌 카드 뭉치 두 개를 선물로 받았습니다. 코니는 다음과 같은 규칙으로 카드에 적힌 단어들을 사용해 원하는 순서의 단어 배열을 만들 수 있는지 알고 싶습니다.

- 원하는 카드 뭉치에서 카드를 순서대로 한 장씩 사용합니다.
- 한 번 사용한 카드는 다시 사용할 수 없습니다.
- 카드를 사용하지 않고 다음 카드로 넘어갈 수 없습니다.
- 기존에 주어진 카드 뭉치의 단어 순서는 바꿀 수 없습니다.
</br>
예를 들어 첫 번째 카드 뭉치에 순서대로 ["i", "drink", "water"], 두 번째 카드 뭉치에 순서대로 ["want", "to"]가 적혀있을 때</br>
["i", "want", "to", "drink", "water"] 순서의 단어 배열을 만들려고 한다면, 첫 번째 카드 뭉치에서 "i"를 사용한 후</br>
두 번째 카드 뭉치에서 "want"와 "to"를 사용하고 첫 번째 카드뭉치에 "drink"와 "water"를 차례대로 사용하면 원하는 순서의 단어 배열을 만들 수 있습니다.
</br>
문자열로 이루어진 배열 cards1, cards2와 원하는 단어 배열 goal이 매개변수로 주어질 때, cards1과 cards2에 적힌 단어들로 goal를 만들 있다면 "Yes"를, 만들 수 없다면 "No"를 return하는 solution 함수를 완성해주세요.

</br>

### 제한사항
- 1 ≤ cards1의 길이, cards2의 길이 ≤ 10
  - 1 ≤ cards1[i]의 길이, cards2[i]의 길이 ≤ 10
  - cards1과 cards2에는 서로 다른 단어만 존재합니다.
- 2 ≤ goal의 길이 ≤ cards1의 길이 + cards2의 길이
  - 1 ≤ goal[i]의 길이 ≤ 10
  - goal의 원소는 cards1과 cards2의 원소들로만 이루어져 있습니다.
- cards1, cards2, goal의 문자열들은 모두 알파벳 소문자로만 이루어져 있습니다.

</br>

### 출력
|cards1|cards2|goal|result|
|---|---|---|---|
|["i", "drink", "water"]	|["want", "to"]	|["i", "want", "to", "drink", "water"]|	"Yes"|
|["i", "water", "drink"]|	["want", "to"]	|["i", "want", "to", "drink", "water"]|	"No"|


📍**입출력 예 #1**
- 본문과 같습니다.
</br>

📍**입출력 예 #2**

cards1에서 "i"를 사용하고 cards2에서 "want"와 "to"를 사용하여 "i want to"까지는 만들 수 있지만</br>
"water"가 "drink"보다 먼저 사용되어야 하기 때문에 해당 문장을 완성시킬 수 없습니다. 따라서 "No"를 반환합니다.
</br>
</br>

## 📌 나의 풀이
~~~python
from collections import deque

def solution(cards1, cards2, goal):
    answer = 0
    
    q1 = deque(cards1)
    q2 = deque(cards2)
    
    
    
    for i in goal:
        
        #step1 :  각 카드에서 맨 왼쪽에 있는 대상 가르키기
        if len(q1) > 0 :
            q1_c = q1.popleft()

        if len(q2) > 0:
            q2_c = q2.popleft()
            
            
        #step2 : goal 단어가 각 카드에서 빼온 대상과 일치하는지 확인
        ### 무조건: 각 카드에 첫번째 요소끼리 비교해야함 (조건에서 카드를 사용하지 않으면 다음 카드로 넘어갈 수 없기 때문)
        
        if i == q1_c:
            answer += 1
            q2.insert(0, q2_c) # 대상이 아니것은 다시 원래 큐 리스트로 넣어줘야함
            
        elif i == q2_c:
            answer += 1
            q1.insert(0, q1_c)
            
        else:
            return "No"

        
    
    if answer == len(goal):
        return "Yes"
~~~

### 💡문제 분석
- 각 카드에 첫번째 요소를 뽑아서 i와 같은 단어인지 확인
  - q1의 요소와 같다면, q2에서 뽑은 원소 다시 원래대로 위치시키고
  - q2의 요소와 같아면, q1에서 뽑은 원소 다시 원래대로 위치시키기
  - **두 카드 모두 해당 되지 않은 단어라면... 바로 "No" 리턴**
    - 조건에서 카드를 사용하지 않으면 다음 카드로 넘어갈 수 없다고 했기 때문
