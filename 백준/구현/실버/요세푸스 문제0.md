## 📌 문제 정의
요세푸스 문제는 다음과 같다. </br>
1번부터 N번까지 N명의 사람이 원을 이루면서 앉아있고, 양의 정수 K(≤ N)가 주어진다.</br>
이제 순서대로 K번째 사람을 제거한다. 한 사람이 제거되면 남은 사람들로 이루어진 원을 따라 이 과정을 계속해 나간다.</br>
이 과정은 N명의 사람이 모두 제거될 때까지 계속된다. 원에서 사람들이 제거되는 순서를 (N, K)-요세푸스 순열이라고 한다.</br>
예를 들어 (7, 3)-요세푸스 순열은 <3, 6, 2, 7, 5, 1, 4>이다.</br>
N과 K가 주어지면 (N, K)-요세푸스 순열을 구하는 프로그램을 작성하시오.
</br>
</br>

### 입력
첫째 줄에 N과 K가 빈 칸을 사이에 두고 순서대로 주어진다. (1 ≤ K ≤ N ≤ 1,000)

### 출력
예제와 같이 요세푸스 순열을 출력한다.

## 📌 나의 풀이
~~~python
n, k = map(int, input().split())
lst= []
for i in range(1, n+1):
    lst.append(i)

people = 0
result = []

for _ in range(n):
    people = (people + (k -1)) % len(lst) #계속 변화하는 리스트의 길이에서 k-1 인덱스 위치의 사람을 삭제하는 거임...
    result.append(lst.pop(people))


# 출력
print('<',end='')
for i in range(0, len(result)-1):
    print(result[i], end=', ')
print(result[n-1], end='')
print('>')
~~~
### 💡문제 분석
- 인덱스 위치를 기준으로 해당 위치에 번호를 출력
  - (k-1)의 위치를 먼저 출력하는데, 이게 변화하는 리스트 길이에 따라 순서가 달라짐
    - **(현재 인덱스 위치에서 + (k-1)번째) % 변화하는 리스트 길이**
~~~
1회차) people = 0일때(현재 인덱스 위치), (0 + 2) % 7 = 2 ; 인덱스 2에 해당하는 값 제거
1회차 이후 변화한 리스트 결과) [1,2,4,5,6,7]

2회차) people = 2일때, (2 + 2) % 6 = 4
2회차 이후 변화한 리스트 결과) [1,2,4,5,7]

3회차) people = 4일때, (4 + 2) % 5 = 1
3회차 이후 변화한 리스트 결과) [1,4,5,7]
~~~
