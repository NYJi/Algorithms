# 📖 백준 2839번) 설탕 배달
## 📌 문제 정의
상근이는 요즘 설탕공장에서 설탕을 배달하고 있다. 상근이는 지금 사탕가게에 설탕을 정확하게 N킬로그램을 배달해야 한다. 
설탕공장에서 만드는 설탕은 봉지에 담겨져 있다. 봉지는 3킬로그램 봉지와 5킬로그램 봉지가 있다.
상근이는 귀찮기 때문에, 최대한 적은 봉지를 들고 가려고 한다.
예를 들어, 18킬로그램 설탕을 배달해야 할 때, 3킬로그램 봉지 6개를 가져가도 되지만, 
5킬로그램 3개와 3킬로그램 1개를 배달하면, 더 적은 개수의 봉지를 배달할 수 있다.
상근이가 설탕을 정확하게 N킬로그램 배달해야 할 때, 봉지 몇 개를 가져가면 되는지 그 수를 구하는 프로그램을 작성하시오.

### 입력
첫째 줄에 N이 주어진다. (3 ≤ N ≤ 5000)

### 출력
상근이가 배달하는 봉지의 최소 개수를 출력한다. 만약, 정확하게 N킬로그램을 만들 수 없다면 -1을 출력한다.
</br>

## 📌 나의 풀이
~~~python
n = int(input())
rest_sugar = n

cnt = 0


#최소 봉지 구하기
while rest_sugar >= 0: # if문을 무조건 지나야 cnt를 출력 가능 따라서, 0인 경우에 if 조건에 들어가게 해야함

    if rest_sugar % 5 == 0:
        cnt += (rest_sugar // 5)
        print(cnt)
        break

    rest_sugar -= 3
    cnt += 1

else:
    print(-1)

~~~

### 💡문제 분석
- **# while rest_sugar > 0 으로 할 경우에는 rest가 0인 경우에 else문의 -1이 출력(정상적인 계산으로 0인 된건데도 -1이 되어버릴 수 있으니 >= 0으로 해야함)**
~~~
📍n = 8인 경우,
(1회차) 8 - 3 = 5
(2회차) 5 % 5 == 0이니 cnt += (5//5 = 0)을 해주고 나서 cnt 출력

📍n = 4인 경우,
(1회차) 4 - 3 = 1
(2회차) 1 - 3 = -2
(3회차) whlie -2 >= 0 --> False 니깐 -1 출력
~~~

❗**여기서 주의**❗
- 근데, if > 0으로 하게 되면 rest가 0이 되는 즉시, while > 0이 False가 되서 else문인 -1가 실행됨
~~~
📍n = 3인 경우,
(1회차) 3 - 3 = 0
(2회차) 0 % 5 == 0 --> True
        cnt += (0//5 = 0)이어서 기존 cnt += 0인 최종 cnt = 1 출력됨

~~~
