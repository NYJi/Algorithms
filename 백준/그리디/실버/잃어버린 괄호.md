# 📖 백준 1541번) 잃어버린 괄호
## 📌 문제 정의
세준이는 양수와 +, -, 그리고 괄호를 가지고 식을 만들었다. 그리고 나서 세준이는 괄호를 모두 지웠다.
그리고 나서 세준이는 괄호를 적절히 쳐서 이 식의 값을 최소로 만들려고 한다.
괄호를 적절히 쳐서 이 식의 값을 최소로 만드는 프로그램을 작성하시오.

### 입력
첫째 줄에 식이 주어진다. 식은 ‘0’~‘9’, ‘+’, 그리고 ‘-’만으로 이루어져 있고, 가장 처음과 마지막 문자는 숫자이다.
그리고 연속해서 두 개 이상의 연산자가 나타나지 않고, 5자리보다 많이 연속되는 숫자는 없다.
수는 0으로 시작할 수 있다. 입력으로 주어지는 식의 길이는 50보다 작거나 같다.

### 출력
첫째 줄에 정답을 출력한다.</br>

## 📌 나의 풀이
~~~python
expression = list(input().split('-'))

# 각 요소에 대한 처리
for idx, i in enumerate(expression):
    num = list(map(int, i.split('+'))) #i요소에 + 있는 식 대상으로 쪼개서 합쳐줌
    expression[idx] = (sum(num))


answer = expression[0] #초기화 (처음요소 항상 숫자)

if len(expression) > 1: # 만약 식이 처음요소만 들어있지 않으면 (처음요소 - 나머지 요소 - 나머지요소...반복)

    for i in range(1, len(expression)):
        answer -= expression[i]
    print(answer)

else: # 식에 처음요소만 있으면 무조건 '+' (조건: 처음요소는 항상 숫자 (연산자X))

    print(answer)

~~~

### 💡문제 분석
- 처음 요소는 항상 숫자이다. (-부호로 시작하지 않음)

- '-' 부호로 쪼갠 후 각 요소에서 + 식 가진 요소 처리를 위한 '+' 부호로 쪼갠 후 합 구하기
  - 각 요소 자리에 해당 합을 넣어주기

- 모든 처리가 끝난 리스트에는 '-' 부호로 쪼갠 요소로 채워져 있으니 각 요소를 이제 빼주기
  - **단! 처음 요소를 제외한 나머지만 빼주기**
